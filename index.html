Welcome to a fork of Foodle

This page is related to a fork of Foodle software developed by UNINETT.

I've decided to fork this piece of software in order to support:

  • PHP 5.4 -- the version available in Debian GNU/Linux (a.k.a. Wheezy),
  • Czech language.

Depending on my employer, I'm going to maintain it for our usage and release all the changes I've written. Feel free to fork, contribute or ask any question.

To clone my Foodle repository using HTTPS protocol, use the following command:

$ git clone https://github.com/JanOppolzer/Foodle.git

Current development

Nowadays, there are two main branches, i.e. master and cesnet. The first branch contains all changes ready for sending upstream. The second branch contains changes required for our deployment; however, some of them should be tweaked and put upstream. It depends on time, but I believe I'll be able to improve the master branch, eventually.
